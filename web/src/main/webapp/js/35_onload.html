DOM(Document Object Model) - onload Event Listener

DOM : document 객체를 기반으로 제공되는 HTML 태그에 대한 자바스크립트 객체 - Node 객체(Element 객체)

브라우저로 웹프로그램을 요청하여 실행결과를 응답받아 브라우저(document 객체)에 출력

브라우저는 브라우저 엔진으로 응답받은 HTML 문서를 읽어 DOM Parser를 사용해 HTML 문서의 태그를 Dom Tree로 생성하여 CSS 스타일을 적용하여 브라우저에 출력 처리

자바스크립트를 사용해 태그에서 이벤트가 발생할 경우 DOM Tree에 존재하는 태그(Dom 객체)를 조작하여 동적인 페이지 구현 가능 - DHTML(Dynamic HTML)

Dom Tree가 완성된 후 동적인 페이지 구현 가능 - onload 이벤트 리스너 속성 사용
